# Mohammad Abotargah - Professional Portfolio

## Overview
This is the source code for the personal professional branding website of **Mohammad Abotargah**, a Senior Operations Professional & AI Integration Specialist.

The website showcases his 15+ years of industrial excellence at SABIC and his transition into AI-driven project management.

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Vanilla CSS (Industrial Modern Design System)
- **Fonts:** Outfit (Headings), Inter (Body)
- **Assets:** Custom AI-generated industrial imagery

## Key Features
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile.
- **Glassmorphism UI:** Modern, sleek interface with glass effects.
- **Interactive Sections:**
  - Hero Section with Trust Bar
  - Professional Expertise Cards
  - Project Portfolio with Case Studies
  - About Me Narrative
  - Contact Information

## How to Run Locally

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/mohammad-abotargah-portfolio.git
    cd mohammad-abotargah-portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Build for production:
    ```bash
    npm run build
    ```

## Deployment
This project is optimized for deployment on Netlify.

1.  Connect your GitHub repository to Netlify.
2.  Set the build command to `npm run build`.
3.  Set the publish directory to `dist`.
